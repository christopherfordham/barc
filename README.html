BARC (BA Roster Check) — Render Deployment Guide

  1. Create a free account at render.com.
  2. Click New → Web Service, then choose Upload a folder or ZIP.
  3. Select this ZIP and upload the backend/ folder (it contains the entire app: API + PWA).
  4. Render will detect Python. Use the defaults in render.yaml:
    • Build Command: pip install -r requirements.txt
    • Start Command: uvicorn main:app --host 0.0.0.0 --port $PORT
  5. Wait for the build to complete. You'll get a URL like https://barc.onrender.com.
  6. Open that URL on your phone. Tap “Add to Home Screen” to install the PWA.

Using the eMaestro Upload

Use the uploader card to select an XML like FORMCOCT25Maestro.xml. The server parses duties, then automatically runs the rule checks and shows a JSON summary and details. This is a permissive parser that looks for common eMaestro nodes (RosterDuty, Flight/Leg, STD/STA UTC attributes). If your export format differs, share a sample and we’ll tune the parser.
